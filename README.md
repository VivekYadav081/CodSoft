# Codsoft

Codsoft is a collection of three beginner-friendly web projects built with HTML, CSS, and JavaScript.

## Projects

### 1. Portfolio Website
A personal portfolio website that showcases skills, projects, resume download, and contact details.

Folder: `portfolio`

### 2. Gamezone Landing Page
A responsive landing page design for a gaming brand with hero visuals, navigation, and featured game cards.

Folder: `landingpage`

### 3. Calculator
A simple calculator app for basic arithmetic operations with a clean UI.

Folder: `Calculator`

## Usage

Each project is a static website. Open the `index.html` file inside the corresponding folder in a web browser.

Example:
- `portfolio/index.html`
- `landingpage/index.html`
- `Calculator/index.html`

## Folder Structure

- `Calculator/` — calculator HTML, CSS, JavaScript files
- `landingpage/` — game landing page HTML and CSS files
- `portfolio/` — portfolio HTML, CSS, and asset files

## Notes

- These projects are suitable for learning web development fundamentals.
- Feel free to customize the styles, content, and layout to match your own preferences.
