# Portfolio Website

A personal portfolio website built with HTML and CSS to showcase skills, projects, and contact details.

## Overview

This portfolio website presents a simple and modern profile page with sections for an introduction, skills, projects, resume download, and contact information.

## Features

- Personal header with name and role
- About section with profile image and introduction
- Skills section with interactive skill cards
- Projects section showcasing web projects
- Resume download link
- Contact section with email and phone details

## Files

- `index.html` — Main portfolio page structure
- `style.css` — Styling and layout for the portfolio
- `asset/Vivek.jpeg` — Profile image used in the About section
- `asset/Vivek_Resume.pdf` — Resume file for download

## Usage

1. Open `index.html` in a web browser.
2. Review the profile sections, skills, and projects.
3. Click the resume link to download the PDF.

## Notes

- This is a static portfolio site suitable for personal branding and job applications.
- You can customize the content, colors, and layout to match your own style.
